## Installation

### Maven

```xml
<dependency>
  <groupId>ink.literate</groupId>
  <artifactId>crowous</artifactId>
  <version>0.1.0</version>
</dependency>
```

### Gradle (Kotlin)

```kotlin
implementation("ink.literate:crowous:0.1.0")
```

### Gradle

```groovy
implementation 'ink.literate:crowous:0.1.0'
```
